# sindredash
> Fighting back against modularity - a [@sindresorhus](https://github.com/sindresorhus) utility suite.


## use
```sh
$ npm install --save sindredash
```
```js
var sindredash = require("sindredash")
// sindredash.hasDom()
// sindredash.isHtml()
// sindredash.compareUrls()
// ...
```
